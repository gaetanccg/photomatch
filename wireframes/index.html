Légende des conventions

  • Bordures pointillées : zones cliquables / interactives

User Flows

Flow Client : Trouver un photographe

Accueil → Recherche → Profil photographe → Créer projet → Envoyer demande → Réponse acceptée → Mission → Laisser avis

Flow Photographe : Recevoir une mission

Dashboard → Notification demande → Voir détail → Accepter + Prix → Mission réalisée → Recevoir avis
